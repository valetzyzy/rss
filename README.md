## Requirements

- [Docker & docker-compose](https://www.docker.com/)
- [node.js](https://nodejs.org/en/)

## Install

- `clone progect`
- `cd project-folder`
- `npm i`
- `npm run build`
- `docker-compose up -d`
- `docker exec -it riga_app_1 bash`
- `composer install`
- `php artisan key:generate`
- `php artisan jwt:secret`
- `php artisan migrate`
- Go to [localhost:8035](http://localhost:8035)
